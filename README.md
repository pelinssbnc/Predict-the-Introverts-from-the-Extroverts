# 🧠 İçedönük / Dışadönük Kişilik Tahmini

Davranışsal veriler kullanarak bireylerin
kişilik tipini tahmin eden sınıflandırma projesi.

---

## 📌 Genel Bakış

Bu projede sosyal davranış örüntüleri ve kişisel tercihler analiz edilerek
bireylerin içedönük (introvert) mi yoksa dışadönük (extrovert) mi olduğu
makine öğrenmesi modelleriyle sınıflandırılmaktadır.

Temel hedefler:
- Kişilik tipini belirleyen temel özellikleri analiz etmek
- İkili sınıflandırma modelleri geliştirmek
- Doğruluk, F1-Score ve ROC-AUC ile değerlendirmek
